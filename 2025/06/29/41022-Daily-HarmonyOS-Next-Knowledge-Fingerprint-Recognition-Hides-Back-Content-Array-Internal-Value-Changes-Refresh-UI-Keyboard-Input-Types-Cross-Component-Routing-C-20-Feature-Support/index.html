3. HarmonyOS keyboard InputType support issue?

Does the keyboard input type only support ASCII characters?

Currently, there is no separate keyboard input type. An input filter (inputFilter) can be set through a regular expression. Input that matches the expression is allowed to be displayed, and input that does not match is filtered. Only single-character matching is supported, and string matching is not supported. For reference, please see the link below: https://gitee.com/openharmony/docs/blob/master/zh-cn/application-dev/reference/apis-arkui/arkui-ts/ts-basic-components-textinput.md#inputfilter8
