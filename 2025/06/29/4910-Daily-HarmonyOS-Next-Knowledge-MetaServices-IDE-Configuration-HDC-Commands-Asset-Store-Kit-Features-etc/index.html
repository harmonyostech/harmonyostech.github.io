1. How to obtain the user’s current location address information in a MetaService?

The location information obtained in a MetaService does not return the address, only latitude and longitude. The API for reverse geocoding latitude and longitude to an address is not supported in MetaServices. Here’s how to obtain the user’s address information:

Use MapKit to get address information based on latitude and longitude: 

let requestInfo: geoLocationManager.CurrentLocationRequest = {
  'priority': geoLocationManager.LocationRequestPriority.FIRST_FIX, 
  'scenario': geoLocationManager.LocationRequestScenario.UNSET,
  'maxAccuracy': 0
};

let locationChange = async (err: BusinessError, location: geoLocationManager.Location): Promise<void> => {
  if (err) {
    console.log('locationChanger: err=' + JSON.stringify(err));
  }
  if (location) {
    console.log('locationChanger: location=' + JSON.stringify(location));
    let position: mapCommon.LatLng = {
      latitude: location.latitude,
      longitude: location.longitude
    };
    try {
      const res = await site.nearbySearch({
        location: position
      });
      console.log(JSON.stringify(res));
    } catch (e) {
      console.log(e);
    }
  }
};

try {
  geoLocationManager.getCurrentLocation(requestInfo, locationChange);
} catch (err) {
  console.error("errCode:" + (err as BusinessError).code + ", errMessage:" + (err as BusinessError).message);
}

2. Issue with file and folder sorting in the HarmonyOS development tool’s Resource Explorer?

The file and folder sorting in the Resource Explorer has changed to alphabetical order. How to revert to the initial default sorting?

  1. Press Shift+Shift on the keyboard.
  2. Click to search for Tree Appearance, select the first matching item to modify the sorting.

3. The IDE is configured for automatic signing, and the signature file exists, but running the app prompts “No signature file”?

  1. Open Project Structure > Signing Config to view the configured signature file directory.
  2. Delete the original signature directory files, reselect Automatically generate signature, and sign again.

4. How to install hdc commands for HarmonyOS?

To debug WebView content, hdc (OpenHarmony Device Connector) commands are required to connect to devices or emulators. How to download and install the latest hdc command-line tool?

  • hdc Overview: hdc is a command-line tool for developers to debug real devices.
  • Location: The hdc tool is located in {SDK directory}/toolchains.
  • Environment Variable Configuration: Refer to https://developer.huawei.com/consumer/cn/doc/harmonyos-guides-V5/hdc-V5.

5. What are the features of Asset Store Kit?

After manually migrating (cloning) phone data to another device, will data in the Asset Store Kit be synchronized to the new device?

Key features of Asset Store Kit:

  1. Data is inaccessible to other apps, only accessible by the business that wrote the critical assets.
  2. No cross-device synchronization is supported.
  3. Supported devices: phones, tablets, and PCs.

Conclusion: Data in Asset Store Kit cannot be synchronized to other devices.
