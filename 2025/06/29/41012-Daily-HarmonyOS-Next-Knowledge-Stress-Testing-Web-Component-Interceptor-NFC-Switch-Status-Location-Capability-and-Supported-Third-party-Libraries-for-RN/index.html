1. Does HarmonyOS’ Wukong support running Python scripts for stress or regular testing?

  1. Python scripts can call the hdc command to execute hdc shell wukong XXX.
  2. Wukong only supports stability stress testing. For general testing, it is recommended to use the arkxtest testing framework.

2. How to handle custom WebHeader during page navigation in Web components?

If many businesses are implemented using H5, the httpHeader during page loading may include information such as user details and the current app theme. In HarmonyOS, the header can be added via loadUrl for the first webpage load, but it cannot be added during subsequent page navigations. What solutions are available for this scenario? Attempts to intercept requests using onInterceptRequest and send requests with headers via HTTP have been ineffective.

For setting webHeader when using webview in ArkTS, refer to: https://developer.huawei.com/consumer/cn/doc/harmonyos-references-V5/js-apis-webview-V5#geolocationpermissions

Requests can be intercepted using the web component’s onInterceptRequest method.
Sample code: 

import web_webview from '@ohos.web.webview';
@Entry
@Component
struct WebComponent {
  controller: web_webview.WebviewController = new web_webview.WebviewController();
  build() {
    Column() {
      Web({ src: $rawfile('i.html'), controller: this.controller })
        .onInterceptRequest((event) => {
          if (event) {
            console.log('urldddd:' + event.request.getRequestUrl());
          }
          return null;
        })
    }
  }
}

3. How to determine the switch status of the system NFC in HarmonyOS?

The current NFC status can be queried, and changes in the NFC status can be monitored. Refer to: https://developer.huawei.com/consumer/cn/doc/harmonyos-references-V5/js-apis-nfccontroller-V5#nfcstate

Testing verifies that controller.isNfcOpen() returns the correct value.
Demo: 

import controller from '@ohos.nfc.controller';

@Entry
@Component
struct TestNetConnectionPage {
  @State monitorNfcState: string = '';
  @State queryNfcState: string = '';
  @State isNfcOpen: boolean = false;

  aboutToAppear(): void {
    if (canIUse("SystemCapability.Communication.NFC.Core")) {
      controller.on("nfcStateChange", state => {
        this.monitorNfcState = state.toString();
      });
    }
  }

  build() {
    Row() {
      Column() {
        Row() {
          Text('Monitor NFC status:').width('50%');
          Text(this.monitorNfcState).width('50%');
        }.alignSelf(ItemAlign.Start);

        if (canIUse("SystemCapability.Communication.NFC.Core")) {
          Row() {
            Button('Query NFC status:').width('50%').onClick(() => {
              this.queryNfcState = controller.getNfcState().toString();
            });
            Text(this.queryNfcState).width('50%');
          }.alignSelf(ItemAlign.Start);
        }

        if (canIUse("SystemCapability.Communication.NFC.Core")) {
          Row() {
            Button('Check NFC switch:').width('50%').onClick(() => {
              this.isNfcOpen = controller.isNfcOpen();
            });
            Text(this.isNfcOpen + '').width('50%');
          }.alignSelf(ItemAlign.Start);
        }
      }
      .width('100%')
    }
    .height('100%')
  }
}
